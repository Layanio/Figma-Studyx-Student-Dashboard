# StudyX - Student Dashboard (Figma Design )

A simple student dashboard  UI designed in Figma.

The project includes login screens , a home dashboard , course cards , agenda section , and event updates.

## Features

- Login screen
- Home dashboard
- Search bar
- Latest Updates section
- Agenda
- Courses
- Events
- Projects notes

## Screens Included

- Welcome / Login
- Dashboard
- Courses
- Updates
- Events
